Dunya is a research prototype developed in CompMusic, a research project that studies several world music traditions from an information technologies point of view. It has been developed to evaluate the results of the research, mainly to be able to get feedback from users to further improve the technologies being developed.

The information shown has been gathered from different sources and we have tried our best to get accurate data and to identify its provenance whenever possible.

Developers

Dunya offers an API that allows other applications to access the collections and data that we computed as part of the CompMusic project. For more information go to the developers page.
